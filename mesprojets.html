SAE 24 / 23 / 15  ·  Collaboration Thales Alenia Space

Système embarqué & site web de capture d'images

Réalisé

Contexte

Dans le cadre d'un projet réalisé en collaboration avec Thales Alenia Space, l'objectif était de concevoir un système embarqué capable de capturer des images de test pour un banc avionique. Le système reposait sur un Raspberry Pi 3 hébergeant une interface web permettant de piloter le matériel et de gérer les données capturées.

Développement web & backend

  • Création des différentes pages de l'interface utilisateur
  • Scripts d'automatisation de capture et de traitement d'images
  • Mise en place d'une base de données pour stocker tests et images

Intégration matérielle

  • Câblage et raccordement des composants électroniques sur le Raspberry Pi 3
  • Configuration des périphériques électroniques
  • Système visuel via LED pour afficher l'état du système

Hébergement & configuration

  • Configuration du Raspberry Pi comme serveur web local
  • Déploiement de l'interface sur le réseau du banc de test
  • Administration système sous Linux (Raspberry Pi OS)

Technologies & compétences

Python HTML / CSS PHP SQL Raspberry Pi 3 🐧 Linux Base de données Électronique embarquée Hébergement web local Travail en équipe Cahier des charges
SAE 25  ·  Découverte du monde professionnel

Interview professionnelle & découverte du monde du travail

Réalisé

Contexte

Dans le cadre d'un projet réalisé en équipe, nous avons mené une interview avec un alternant de 2ème année travaillant comme IT Technician. L'objectif était de mieux comprendre le fonctionnement du monde professionnel dans le domaine de l'informatique et des réseaux, ainsi que les compétences attendues en entreprise. Cette rencontre nous a permis d'avoir un retour concret sur les missions quotidiennes, les responsabilités du métier et les qualités importantes pour évoluer dans le secteur.

Préparation de l'interview

  • Recherche d'informations sur le métier d'IT Technician
  • Préparation des questions en équipe pour couvrir différents aspects
  • Organisation de l'entretien et répartition des rôles

Réalisation de l'entretien

  • Échange sur le parcours et l'expérience professionnelle
  • Découverte des missions liées au métier d'IT Technician
  • Discussion sur les compétences techniques et humaines attendues

Analyse & restitution

  • Prise de notes et synthèse des informations recueillies
  • Mise en commun des réponses avec l'équipe
  • Réflexion sur les compétences à développer pour l'insertion pro

Compétences mobilisées

Communication professionnelle Travail en équipe Organisation de projet Prise de parole Analyse & synthèse Découverte du monde pro Métiers informatique & réseaux Esprit d'analyse Curiosité professionnelle
